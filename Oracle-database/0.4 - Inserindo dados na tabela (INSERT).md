
Para incluir dados, vamos criar um novo **_script_** vazio (ou planilha SQL) para **digitar os comandos** SQL de inclusão. Para isso, vá ao menu superior em "Arquivo > Novo" ou aperte "Ctrl + N".

Na caixa "Nova Galeria", selecione "Arquivo de Banco de Dados" na lateral esquerda intitulada "Categorias". Em seguida, clique em no botão "OK" do canto inferior direito.

Ao fazermos isso, abre uma caixa de diálogo "<font color="#ff0000">Criar Arquivo SQL</font>" aonde devemos colocar um título no campo "<font color="#ff0000">Nome do Arquivo</font>". 

## Inserindo registros na tabela

Tabela TB_PRODUTOS : 
```SQL
CREATE TABLE TB_PRODUTOS
(
    CODIGO_PRODUTO VARCHAR2(20),
    NOME_PRODUTO VARCHAR2(150),
    EMBALAGEM VARCHAR2(50),
    TAMANHO VARCHAR2(50),
    SABOR VARCHAR2(50),
    PRECO_LISTA NUMBER
);
```

Inserindo dados : 
```SQL
INSERT INTO TB_PRODUTOS (

)
```

Para inserir dados em uma tabela usamos o comando #INSERT que significa #Inserir 
* Devemos especificar o local da inserção na mesma linha. Neste exemplo vamos inserir na TB_PRODUTOS. Escrevemos ``INTO TB_PRODUTOS`` 
* Precisamos <font color="#ff0000">abrir e fechar parênteses</font> para delimitar o comando.
* Entre os parênteses "()" do comando, vamos escrever os nomes das colunas da tabela

```SQL
INSERT INTO TB_PRODUTOS (
PRODUTO, NOME, EMBALAGEM, TAMANHO, SABOR, PRECO_LISTA)
```

* Vamos digitar os nomes de todos os campos da tabela em uma mesma linha, separados por uma vírgula " <font color="#ff0000">,</font> ".
### Inserindo os valores de nossa tabela
* Devemos escrever o comando ``VALUES()``  para colocarmos os valores que queremos adicionar.
* A ordem dos valores deve corresponder a mesma ordem dos nossos campos.
* Ou seja, o <font color="#ff0000">primeiro dado</font> de `VALUES` <font color="#ff0000">deve ser para o campo PRODUTO</font>

```SQL
INSERT INTO TB_PRODUTOS
(PRODUTO, NOME, EMBALAGEM, TAMANHO, SABOR, PRECO_LISTA)
VALUES
('1040107', 'Light - 350 ml - Melancia', 'Lata', '350 ml', 'Melancia', 4.56);
```
