* DDL 
	* CREATE = Cria uma base de dados de tabela.
	* ALTER =  Que altera a propriedade dos componentes do banco.
	* TRUNCATE = Apaga de forma definitiva.
	* DROP = Ele apaga os componentes da estruturas de dados.
* DML 
	* INSERT = Incluir dados em nossa tabelas.
	* UPDATE = Altera os dados da nossa tabela.
	* DELETE = Apaga os dados da minha tabela.
	* LOCK = Gerencia as ocorrências de dados da tabela.
* DCL 
	Comandos que nos permitem administrar o nosso banco de dados.
	* COMMIT = Salva o estado do banco de forma definitiva no disco
	* ROLLBACK = Que retorna o estado de um banco de dado relacional a partir de um ponto salvo previamente.
	* SAVEPOINT = Salva esse ponto prévio no banco de dados.
