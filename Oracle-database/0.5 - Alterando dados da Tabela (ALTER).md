Devemos usar o comando ``UPDATE``, seguido do nome da tabela e o comando chamado ``SET`` após isso vamos escrever o que queremos alterar.

Neste caso queremos alterar os dados do campo ``EMBALAGEM`` para o o texto ``Lata``

![[Pasted image 20260614191443.png]]

* Já dissemos para o SQL que queremos alterar a características da embalagem de alguém na tabela do produto.  
* Mas <font color="#ff0000">temos que especificar qual o produto que queremos alterar</font>  **1037797**
* Mas para dizer aonde queremos fazer essas alterações devemos usar o ``WHERE``  ainda na mesma linha. 
```SQL
UPDATE TB_PRODUTOS SET EMBALAGEM = 'PET' WHERE PRODUTO = '1037797';
```
* Usamos o campo ``CODIGO_PRODUTO`` que neste caso era o **1037797** neste caso ficaria assim: ``CODIGO_PRODUTO = '1037797'``  
*![[Pasted image 20260614192343.png]]

## Alterando mais de um dado da tabela 
```SQL
UPDATE TB_PRODUTOS SET NOME_PRODUTO = 'Sabor da Montanha - 350 ml - Uva', TAMANHO = '350 ml', PRECO_LISTA = 5.50
WHERE CODIGO_PRODUTO = '1037797';
```
* Ao final, vamos acrescentar novamente qual o produto que terá esses campos atualizados, usando o comando `WHERE` para o campo `PRODUTO` igual à _string_ de texto `1037797`.

## Alterar tipo ou tamanho de uma coluna
```SQL
ALTER TABLE nome_tabelaMODIFY nome_coluna VARCHAR2(100);
```

Exemplo:

```sql
ALTER TABLE TB_CLIENTESMODIFY NOME VARCHAR2(80);
```

## Renomeando uma coluna
```SQL
ALTER TABLE nome_tabelaRENAME COLUMN nome_antigo TO nome_novo;
```

 Exemplo:

```SQL
ALTER TABLE TB_CLIENTESRENAME COLUMN ENDERECO TO LOGRADOURO;
```

## Adicionar uma nova coluna

```SQL
ALTER TABLE nome_tabelaADD nome_coluna TIPO_DADO;
```

 Exemplo:

```SQL
ALTER TABLE TB_CLIENTESADD EMAIL VARCHAR2(100);
```

## Remover uma coluna

```SQL
ALTER TABLE nome_tabelaDROP COLUMN nome_coluna;
```

 Exemplo:

```SQL
ALTER TABLE TB_CLIENTESDROP COLUMN EMAIL;
```
