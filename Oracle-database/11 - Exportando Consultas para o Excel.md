Para exportar o resultado de uma pesquisa SQL em uma tabela Excel, você pode seguir os seguintes passos:

1. Execute sua consulta SQL no Oracle SQL Developer;
    
2. Clique com o botão direito do mouse no resultado da consulta e selecione "Exportar...":
    
    ![Selecionando uma consulta e clicando em 'Exportar' no SQL Developer.](https://i.imgur.com/37P10d3.png)
3. Selecione o formato "Excel 2003+ (xlsx)" e defina o nome do arquivo e seu caminho:
    
    ![Exportação de consulta no SQL Developer](https://i.imgur.com/rGDpa4f.png)
4. Clique em "Próximo" e logo em seguida em "Finalizar". Depois disso, basta aguardar o processo de exportação ser concluído!
