Podemos criar uma tabela em nosso banco de dados usando os comandos SQL 

O comandos **CREATE TABLE** é utilizado para criar uma nova tabela no banco de dados.

Sintaxe: 
```Java 
CREATE TABLE nome_tabela 
(
	CPF VARCHAR2(11),
    NOME VARCHAR2(100),
    ENDERECO1 VARCHAR2(150),
    ENDERECO2 VARCHAR2(150),
    BAIRRO VARCHAR2(50),
    CIDADE VARCHAR2(50),
    ESTADO VARCHAR2(2),
    CEP VARCHAR2(8),
    DATA_NASCIMENTO DATE,
);
```

## Principais tipos de dados 
### NUMBER

Armazena valores numéricos.

```
idade NUMBER
```

Também pode definir precisão:

```
salario NUMBER(10,2)
```

Onde:

- 10 = total de dígitos.
- 2 = casas decimais.

Exemplo:

```
99999999.99
```

---

### VARCHAR2

Armazena textos de tamanho variável.

```
nome VARCHAR2(100)
```

Pode armazenar até 100 caracteres.

---

### CHAR

Armazena textos com tamanho fixo.

```
sexo CHAR(1)
```

Exemplos:

```
MF
```

---

### DATE

Armazena datas.

```
data_nascimento DATE
```

Exemplo:

```
10/06/2026
```
