O campo que possui o <font color="#ff0000">NOT NULL</font> **deve ser preenchido**, caso o campo fique vazio ao rodar ira exibir um erro.

Exemplo: 
```SQL
NOME VARCHAR2(100) NOT NULL
```

podemos também usar desta seguinte forma: 
```SQL
CONSTRAINT NN_CLIENTES_NOME CHECK (NOME IF NOT NULL)
```

Mas o mais usado e da seguinte forma: 
```SQL
NOME VARCHAR2(50) NOT NULL;
```
