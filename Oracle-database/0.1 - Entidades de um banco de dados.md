A entidade mais importante se chamada **tablespace** significa are de tabelas. Dentro da tablespace posso ter diversas tabelas. 
## Tablespace 

As **tablespace** possuem diversas entidades
* Tabelas - Aonde os dados ficam armazenados (informações são guardadas)
	Uma tabela possui uma estrutura que contém **linhas** e **colunas**
	![[Pasted image 20260613181441.png]]

- Números de linhas posso ter quantas eu quiser, porém tem que caber dentro do espaço de disco disponível
- Número de colunas são limitados. Quando eu definir não posso modificar.
- Colunas são chamadas de **CAMPOS**.
- Linhas chamadas de **REGISTROS** OU **LINHAS** 
![[Pasted image 20260613181830.png]]

## Chaves primarias ( #PRIMARYKEY)
CHAVE PRIMÁRIA (NÃO É OBRIGATÓRIO)
Colunas cujos **valores não podem se repetir** nas linhas da tabela.

## Chaves Estrangeiras ( #FOREINGKEY) Relacionamentos
 * As chaves **estrangeiras fazem as ligações entre os dois campos de duas tabelas diferentes**
 * relacionamento é a **chave estrangeira**, pois faz a **ligação** entre dois campos de duas tabelas diferentes.
 * Tenho o campo 3 da tabela (cliente) e tenho o campo 4 da tabela (pedido) aonde o valor estará associado a um valor de um outro campo.
*  Os dois campos devem ter as mesmas características (**tipos**) caso o campo3 seja String o campo4 deve ser também.

*![[Pasted image 20260613182925.png]]
* Esse campo B, que é o campo de chegada da chave estrangeira, **deve ser obrigatoriamente uma _primary key_**, porque este campo tem que ser único.

## Índice (Fazer Busca)
* O índice é uma tabela a parte que tem coordenadas do endereço daquele dado na tabela.

* **Toda a chave primária vai ter um índice criado**.

* Podemos ter colunas que não sejam chaves primárias mas que tenham índices.

* Pode ser usado para fazer busca em nossa tabela, irei ter que percorrer a tabela de clientes , quais são os clientes que busquei.

## #Trigger

Trigger é uma regra que eu crio no meu banco de dados que vai efetuar alguma coisa quando algo acontecer.

Exemplo: 
Toda vez que alguém incluir algum dado na tabela de vendas de cliente eu tenho que fazer uma sumarização em outra tabela.
