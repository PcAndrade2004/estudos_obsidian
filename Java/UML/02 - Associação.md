 Em UML, associação é um relacionamento entre duas classes que indica que uma classe usa a outra.
### Exemplo Simples
* Classe *Aluno*
* Classe *Curso*
Um aluno pode estar matriculado em um curso 
Então existe uma associação entre Aluno e Curso 

![[Pasted image 20260525185511.png]]

## Em Java como isso Aparece 
A associação normalmente aparece através de: 
* Atributos 
* parâmetros 
* listas 
* Objeto dentro de outra classe
