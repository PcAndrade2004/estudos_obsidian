# Agregação 
A agregação representa uma relação de: 
```Java 
"tem um"
```
mas aonde os objetos podem existir separadamente 

* O objeto *filho* pode existir sem o *pai*
# Exemplo clássico

```
Time ◇-------- Jogador
```

O jogador existe sem o time.

Se o time acabar:

- o jogador continua existindo.

# Símbolo UML

A agregação usa:

```
losango branco ◇
```

Exemplo:

```
Time ◇-------- Jogador
```

O losango fica no lado da classe “dona”.
